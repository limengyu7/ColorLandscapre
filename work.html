大概思路（随时调整）

一．问题：

目的是要从每个像素仅具有1个值（仅亮度）的灰度图像中推断出每个像素（亮度、饱和度和色调）具有3个值的全色图像，（应该是一个非监督式学习模型）我们只能处理大小为256 x 256的图像，所以我们的输入图像大小为256 x 256 x 1（亮度通道），输出的图像大小为256 x 256 x 2（另两个通道）。使用LAB色彩空间（亮度，A和B)

二．工具：

用PyTorch构建和训练搭建的模型。使用torchvision工具，scikit-learn工具，用于在RGB和LAB颜色空间之间进行转换

三．用卷积神经网络CNN,使用SE-Inception-ResNet-v2特征提取器，模型：迁移学习的方法，结合深度神经网络Inception-ResNet-v2设计了一个全自动的着色网络模型，在模型中加入了SENet模块

四．在模型中使用SE-Inception-ResNet-v2网络并从其最后一层获取灰度图像的嵌入，它主要由3部分组成，分别是特征提取模块、融合模块和重建模块

五．在网络中使用欧氏距离函数作为损失函数！

六．在项目进行过程中，全体成员之后进行补充说明。
